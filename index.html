見出し Werckerの自動デプロイがうまくいかないので、暫定で手動デプロイする手順。 cd Sites hugo -t blackburn cd public git add -A git commit git push 参考URL: http://ureta.net/2015/05/hugo-on-windows/ http://tanksuzuki.com/post/hugo-github-pages-2/ http://tanksuzuki.com/post/hugo-github-pages-1/ ローカルで確認する方法 hugo server -t blackburn -D -w highchartsの表示テスト Highchartsを呼び出すテスト。 テンプレートのhead.htmlに以下を追加 <!-- add my sciprt --> <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.2/jquery.min.js"></script> <script src="http://code.highcharts.com/highcharts.js"></script> <script src="/js/highchartssetting.js"></script> テーブルのテスト 画像がうまく貼れていないので要調査。 http://webkaru.net/jquery-plugin/columns/ http://michaeleisenbraun.com/columns/
