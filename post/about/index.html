見出し

Werckerの自動デプロイがうまくいかないので、暫定で手動デプロイする手順。

cd Sites
hugo -t blackburn
cd public
git add -A
git commit
git push

参考URL:

http://ureta.net/2015/05/hugo-on-windows/

http://tanksuzuki.com/post/hugo-github-pages-2/

http://tanksuzuki.com/post/hugo-github-pages-1/

ローカルで確認する方法

hugo server -t blackburn -D -w
